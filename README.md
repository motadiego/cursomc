# Cursomc
Api desenvolvida para controle de um carrinho de compras.

## Technoclogies
 <li>Java 11</li>
 <li>Spring Boot</li>
 <li>Spring Data JPA</li>
 <li>Spring Security</li>
 <li>Autenticação com JWT</li>
 <li>MySql database</li>
 <li>Integração com AWS</li>
 <li>Documentação com Swagger</li>
 <li>Maven</li>
